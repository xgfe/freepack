# Freepack -- a file version free pack manager.
> `freepack` allow to pack free version submodule, package and file.

## Installation
In global mode:

`npm install -g freepack`

In local mode:

`npm install --save-dev freepack`


## Usage
```shell
cd $project_dir

# [Optional] Create freepack.config.js file
# freepack init

freepack
```
